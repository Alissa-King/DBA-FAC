About this project — click to expand or collapse
What is the FAC? The Federal Audit Clearinghouse (FAC) is the government database of Single Audit reports — independent annual audits required for any organization spending $750,000 or more in federal funds in a fiscal year.
What does this project ask? Can we predict which organizations will receive audit findings before their audit closes, using only information already on file? If yes, the model can support risk-based triage and resource allocation.
Why is the test set a future year? Models are trained on earlier years and tested on a future year they have never seen — the same way a real risk-screening tool would operate. A random split would let the model "see the future" and inflate performance scores.
